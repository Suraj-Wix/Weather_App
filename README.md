# 🌦️ Weather App

A responsive Weather Forecasting Web Application built using HTML, CSS, and JavaScript. It provides real-time weather information for any city using the OpenWeatherMap API.

## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Real-time temperature updates
- 💧 Humidity information
- 🌬️ Wind speed details
- ☁️ Weather conditions with icons
- 📱 Fully responsive design
- ⚡ Fast and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Suraj-Wix/Weather_App/weather-app.git
```

2. Navigate to the project folder

```bash
cd weather-app
```

3. Open `index.html` in your browser.

## 🔑 API Setup

1. Create an account on OpenWeatherMap.
2. Generate your API key.
3. Replace the API key in `script.js`.

```javascript
const apiKey = "YOUR_API_KEY";
```
